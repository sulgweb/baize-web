# 📦 Baize 官网

> Baize 官方网站，基于 Next.js 构建的现代化官网项目。

🌐 在线地址：https://baize.plume.vip

---

## ✨ 项目简介

Baize 官网是一个基于 **Next.js** 构建的高性能网站，专注于：

- ⚡ 极致性能（LCP / Core Web Vitals 优化）
- 🧩 模块化组件设计
- 🎨 现代 UI / 响应式布局
- 🚀 SEO 友好
- 🌍 可扩展的国际化支持

---

## 🛠 技术栈

- **框架**：Next.js 16（App Router）
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **构建工具**：Turbopack / Webpack（可选）
- **部署**：Vercel / 自托管
- **其他**：
  - next/image（图片优化）
  - next/font（字体优化）
  - 动态路由 & 按需加载
